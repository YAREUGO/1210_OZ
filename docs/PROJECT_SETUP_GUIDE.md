## 프로젝트 초기 설정 순서

### 1단계: 프로젝트 셋업 (1일차)

#### 1.1 Next.js 프로젝트 생성

```bash
# Next.js 15 App Router로 프로젝트 생성
npx create-next-app@latest my-project --typescript --tailwind --app --no-src-dir

# 패키지 매니저 선택 (pnpm 권장)
cd my-project
pnpm install
```

#### 1.2 필수 패키지 설치

```bash
# UI 컴포넌트
pnpm add shadcn-ui
pnpm dlx shadcn@latest init

# 인증
pnpm add @clerk/nextjs

# 데이터베이스
pnpm add @supabase/supabase-js @supabase/ssr

# 아이콘
pnpm add lucide-react

# 차트 (필요 시)
pnpm add recharts

# 폼 관리 (필요 시)
pnpm add react-hook-form zod @hookform/resolvers
```

#### 1.3 기본 디렉토리 구조 생성

```
my-project/
├── app/                    # Next.js App Router
├── components/
│   ├── ui/                # shadcn 컴포넌트
│   └── ...                # 커스텀 컴포넌트
├── lib/
│   ├── api/              # API 클라이언트
│   ├── types/            # TypeScript 타입
│   └── utils/            # 유틸리티 함수
├── docs/                 # 프로젝트 문서
├── supabase/
│   └── migrations/       # 데이터베이스 마이그레이션
└── public/              # 정적 파일
```
